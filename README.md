# Mino Skincare

Bienvenue sur le projet **Mino Skincare**. C'est un site de démonstration créé pour présenter une marque de cosmétique et proposer une boutique en ligne sur le même site.

## Objectif du projet

Ce projet est conçu comme une application fonctionnelle pour :

- présenter une entreprise de cosmétique (page « vitrine »)
- montrer l'identité de la marque, les produits et les engagements
- proposer un espace d'achat en ligne pour des produits de beauté
- servir de support pratique à un portfolio ou à une présentation à un futur recruteur

## Ce que le site montre

Il s'agit d'un site mixte :

- une partie **vitrine** pour raconter l'histoire de la marque
- une partie **boutique** pour consulter et acheter des produits cosmétiques
- une expérience simple et professionnelle adaptée à une entreprise moderne

## Technologies utilisées

Ce projet utilise les technologies suivantes :

- **Next.js** : framework web pour construire le site et ses pages
- **Tailwind CSS** : bibliothèque pour styliser le site rapidement et proprement
- **Prisma** : outil pour gérer les données et communiquer avec la base de données
- **PostgreSQL** : base de données pour stocker les produits et les commandes
- **Docker** : facilité de mise en place de l'environnement de développement

## Pour qui c'est utile

Ce README est pensé pour des non-codeurs et des recruteurs. Il explique le projet sans entrer dans des détails techniques trop avancés.

- Pour un recruteur : c'est un projet de démonstration de développement web moderne.
- Pour un futur employeur : il montre des compétences en conception de site, intégration de catalogue produit et gestion d'une application full-stack.

## Lancer le projet

Le projet peut être lancé localement. Si tu es développeur, voici les étapes simples :

1. installer les dépendances avec `pnpm install`
2. lancer le site avec `pnpm dev`
3. ouvrir `http://localhost:3000` dans un navigateur

Si tu utilises Docker, le projet peut aussi fonctionner avec la configuration Docker disponible dans le dépôt.

## Fonctionnalités prévues

Voici les fonctionnalités principales que le site propose ou pourrait inclure :

- **Page d'accueil vitrine** : présentation de la marque Mino Skincare, ses valeurs et ses engagements
- **Catalogue de produits** : liste des soins cosmétiques avec descriptions, photos et prix
- **Détails produit** : page individuelle pour chaque produit avec informations complètes
- **Panier d'achat** : ajout de produits au panier et gestion des quantités
- **Processus de commande** : formulaire de commande avec informations client
- **Gestion des utilisateurs** : inscription et connexion pour les clients
- **Design responsive** : adaptation du site aux mobiles et tablettes
- **Intégration de paiement** : système sécurisé pour les achats (à implémenter)

Ces fonctionnalités montrent une application complète, prête pour une utilisation réelle.

Le nom du projet est **Mino Skincare**, une marque fictive de cosmétique. Le site est fait pour servir de base et peut être enrichi avec :
